# Github Finder - With React

**A Github API Project Displaying User Profiles on Search and Also U Can Check Details Related to their Profile and Repos and Followers**

### Installation

#### Step 1

- `clone the repo`

#### Step 2

- `npm install`

#### Step 3

- `npm start`
